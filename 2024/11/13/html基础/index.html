html 基本结构

  • 一个网页由 html 标签包裹
  • head 标签中的内容不回呈现在网页中
  • 页面整体内容放在 body 标签之内

html 标签

  1. 标签又称元素，是 HTML 的基本组成单位
  2. 标签分类：双标签 和 单标签
  3. 标签名不区分大小写，但是推荐小写
  4. 双标签
  5. 单标签
  6. 标签之间的关系： 并列关系、嵌套关系，可用 ‘tab’ 键进行缩进

html 标签属性

  1. 用于给标签添加 附加信息
  2. 书写位置：标签名之后，属性名之间，属性值之间，属性值之间，属性值用引号括
  3. 属性名： 有些特殊属性，只有属性名，没有属性值
  4. 常用属性：
    - id：唯一标识符，用于区分元素，必须唯一
    - class：元素分组，可以重复，多个元素可以同时使用一个class
    - href：超链接地址
    - target：打开方式，_self：当前窗口打开，_blank：新窗口打开
    - src：图片地址
    - alt：图片替换文本，当图片加载失败时，显示alt中的文本
    - type：input类型，text：文本，password：密码，submit：提交，reset：重置，button：按钮
  5. 注意点：
    - 不同的标签有不同的属性；也存在一些通用属性（在任何标签内都可使用）
    - 属性名和属性值不可以乱写，需要符合 w3c 规定
    - 属性名、属性值不区分大小写，但是推荐小写
    - 属性值用双引号包括，也可以使用单引号，还可以不写引号，但是推荐使用双引号
    - 标签中同一属性名最好只出现一次，否则后写的会失效
